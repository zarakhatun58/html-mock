Assignment V 1.0
Requirements
The final deliverable should be a single self-contained HTML file. The page must be responsive and well-structured. It must include 2–3 sections of your choice. Feel free to stray from these requirements creatively so long as it works in a browser.

1. Hero Section:

  • Create a hero banner for the company “CellStart” and product “ChronoNAD+”. The section should include a main headline, a subtext, and a call-to-action button, and 1–2 additional sections.

1-2 Additional Sections

  • Create two to three additional sections underneath the hero section of your choice.
  • These sections should be designed to showcase the product's benefits or features. Use the provided brand data for content inspiration.

Technical Requirements

  • Build the page using either raw HTML and CSS (no frameworks like Bootstrap) or in Shopify's Liquid template format.
  • Construct 1 self-contained mockup.html. Inline CSS/JS and embed images (data URIs or inline SVG). No network/CDN dependencies.
